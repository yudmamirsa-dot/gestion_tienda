Este repositorio contiene la integración básica del proyecto Gestión de Tienda con una API REST, como parte de la Evidencia GA8.

Módulos integrados

Usuarios

Clientes

Tecnologías utilizadas

Frontend: HTML, CSS y JavaScript

Backend (API REST): Node.js + Express

Base de datos: MySQL

Ejecución de la API

Ir a:

gestion_tienda/api_rest


Instalar dependencias:

npm install


Ejecutar servidor:

node server.js


API disponible en:

http://localhost:3000

Endpoints usados
Usuarios

GET /api/usuarios

POST /api/usuarios

Clientes

GET /api/clientes

POST /api/clientes

Evidencias

Las capturas solicitadas están en:

Evidencia_GA8/capturas
